Console-Based Student Management System:

A console-based Student Management System developed in Java that performs CRUD operations to add, view, update, delete, and search student records.

Features:

-> Add, view, update, delete, and search student information

-> Persistent storage using file handling

-> Simple and intuitive command-line interface (CLI)

-> Built using object-oriented programming (OOP) principles

Tech Stack: Java, File Handling, OOP, CRUD Operations, CLI
